    Integrasi Numerik¶

    Pengertian¶

    Dalam integral analitis, integral tentu dari suatu fungsi dihitung dengan cata : yang mana F(x) adalah integral dari f(x) sedemikian hingga F'(x) = f(x). Contohnya : Secara geometris, perhitungan integral tentu analog dengan perhitungan luas daerah di bawah kurva dan diatas sumbu x, sebagaimana diilustrasikan diatas :

    â€‹ Pada kasus dimana fungsi f(x) sulit dicari integralnya secara analitik, digunakan metode integrasi numerik dengan mengacu pada perhitungan luas daerah, jadi, integrasi secara numerik dilakukan jika:

    1. Integral tidak dapat (sangat sulit) diselesaikan secara analitis

    2. Fungsi yang diintegralka tidak diberikan dalam bentuk analitis, tetapi secara numerik dalam bentuk data angka (dalam bentuk tabel).

    Metode integral numerik merupakan integral tertentu yang didasarkan pada hitungan perkiraan. Hitungan perkiraan tersebut dilakukan dengan fungsi polimonial yang diperoleh berdasarkan data tersedia. Metode integrasi numerik yang paling sering digunakan, mulai dari yang paling sederhana hingga yang lebih rumit, diantaranya adalah:

    1. Metode Trapezoid
    2. Metode Simpson
    3. Metode Quadratik Gauss

    Metode Trapezoid¶

    Menurut rumus geometri, luas trapesium adalah jumlah sisi sejajar dikali tinggi. Dalam kaitannya dengan grafik fungsi f(x), dalam koordinat Kartesius, yang diasumsikan berada di atas sumbu x, maka diperoleh rumus metode integrasi trapezoid sebagai berikut:

    Algoritma Trapezoid¶

    1. Definisikan nilai y = f(x)
    2. Menentukan batas bawahnya (x0) dan batas atasnya (xn).
    3. Tentukan jumlah pembagi (n)
    4. Hitung h = (xn - x0) / n
    5. Hitung integrasi = f(x0) + f(xn)
    6. for i=1 : n
    7. Hitung k = x0 + i*h
    8. Hitung integrasi = integrasi + 2 * f(k)

    Implementasi Python¶

    Trapezoid Code

    import numpy as np
import matplotlib.pyplot as plt

def trapesium(x):
    return (x**2)*np.exp(-x)

a = 1.0
b = 10.0
n = 50

dx = (b-a)/(n-1)
x = np.linspace(a,b,n)

sum = 0.0
for i in range(1,n-1):
    sum += trapesium(x[i])

hasil = 0.5*dx*(trapesium(x[0])+2.0*sum+trapesium(x[-1]))
print(hasil)

nd = 1000
xp = np.linspace(a,b,nd)

plt.plot(xp,trapesium(xp))
for i in range(n):
    plt.bar(x[i], trapesium(x[i]),
    align="edge",
    width=0.000001,
    color="grey",
    edgecolor="red")

plt.fill_between(x,trapesium(x),color="aqua")
plt.show()
